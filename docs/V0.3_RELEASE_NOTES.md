# v0.3.0 Release Notes

## Goal
This release focuses on making the updater reliable before adding heavier antivirus features.

## Changes
- Fixed GitHub latest-release version comparison.
- Handles `v0.3.0` versus `0.3.0` correctly.
- Detects the correct release asset name automatically.
- Prefers `ProcessSecurityAuditorAVPro.exe`.
- Falls back to any `.exe` attached to the release.
- Stops treating updater errors as “up to date.”
- Adds `scripts/test_updater.py`.

## Test checklist
1. Set local `VERSION` to `0.2.0`.
2. Build the app.
3. Publish GitHub release `v0.3.0`.
4. Attach `ProcessSecurityAuditorAVPro.exe`.
5. Run the older app and click Check for Updates.
6. Expected result: update available.

## Important
The updater can only detect v0.3.0 after the v0.3.0 GitHub Release is published with an EXE asset attached.
